# E-Notice Board

E-Notice Board is a web-based notice management system developed as a final year project. The system is designed to help institutions and departments share important announcements efficiently and securely across different user roles such as administrators, lecturers, and students.

## Project Description

This project provides a simple and practical solution for posting, viewing, and managing notices in a structured environment. It allows authorized users to publish notices for specific audiences, ensuring that the right information reaches the right group at the right time.

The application demonstrates core web development concepts such as:

- User authentication and role-based access control
- Notice creation, retrieval, and deletion
- Separate dashboards for different user roles
- Secure database-driven operations with PHP and MySQL

## Key Features

- Secure login and signup system
- Role-based access for admin, lecturer, and student users
- Notice board for announcements and updates
- Targeted notices for specific user groups
- Simple and responsive user interface

## Technology Stack

- PHP
- MySQL
- HTML, CSS, and JavaScript
- XAMPP for local development

## Project Setup

1. Install XAMPP and start Apache and MySQL.
2. Create a database named `miracle_notice_board`.
3. Import the SQL script from `schema.sql`.
4. Place the project folder in `htdocs`.
5. Open the project in your browser using:
   `http://localhost/E-Notice Board`

## Folder Structure

- `admin/` - Admin dashboard and admin-specific operations
- `lecturer/` - Lecturer dashboard
- `student/` - Student dashboard
- `api/` - Backend endpoints for notice and user actions
- `config/` - Database configuration
- `includes/` - Shared PHP includes
- `assets/` - CSS and JavaScript files

## Purpose of the Project

This system was created to improve communication within an academic environment by offering a centralized platform for notices and announcements. It highlights the practical use of web technologies in building a secure and user-friendly application for real-world situations.
